## 1. Requirements

* froxlor **2.0+**
* **PHP 7.4+**
* Required PHP extensions:
    * session, ctype, xml, filter, posix, mbstring, curl, gd, gmp, json
* Suggested PHP extensions:
    * bcmath, gnupg, zip
